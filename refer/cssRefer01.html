CSS란?

CSS는 Cascading Style Sheets 의 약자입니다.

CSS는 HTML 요소들이 각종 미디어에서 어떻게 보이는가를 정의하는 데 사용되는 스타일 시트 언어입니다.

HTML이 실제 표시되는 방법(색상, 크기, 폰트, 레이아웃)을 지정하여 콘텐츠를 꾸며주는 정적인 언어입니다.

CSS 선언 방법

내부 스타일(Internal style sheet)

태그가 아닌 html 문서 내에 CSS를 적용하는 방법

<<!DOCTYPE html>
<html lang="ko>
<head>
    <style>
    body { background-color : red; }
      p  { color: blue;  text-decoration: underline; }
    </style>
</head>
<body>
    <p>내부 스타일을 이용하여 스타일을 적용하였습니다.<p>
</body>
</html>

외부 스타일

스타일 시트를 외부 파일로 저장하여 외부 파일을 html페이지로 불러오는 방법

<!DOCTYPE html>
<html lang="ko>
<head>
    <link rel="stylesheet" href="외부 파일이 위치한 경로를 입력 합니다.">
</head>
<body>
</body>
</html>

인라인 스타일

각각의 태그마다 스타일을 적용하는 방법

1. link 태그: 외부 파일과 연결시 사용 되는 태그

2. type 속성: 외부 파일 종류가 어떠한 것인지 정하는것

3. rel 속성: html 파일과 외부 파일 간의 관계를 나타내는것

4. href 속성: 외부 파일의 위치를 입력하는것.

<!DOCTYPE html>
<html lang="ko>
<head>
</head>
<body>
    <p style="color:green; text-decoration:underline">
    인라인 스타일을 이용하여 스타일을 적용하였습니다.</p>
</body>
</html>
