id/class

ID CLASS
한페이지에 하나의 정의로 단,하나의 태그(id="")만 사용할 수 있다 한페이지에 반복적으로 사용되는 스타일을 정의 할 수있다.
로고, 상단메뉴, 하단정보 같은 스타일을 정의할 때 id를 사용한다. id의 속성이 class의 속성보다 우선 순위가 높습니다. 반복적으로 사용되는 스타일에는 class를 이용하여 정의합니다.
선언하는 방법(#) ex) #footer{} 선언하는 방법(.) ex) .center{}

스타일 적용 방법 예시입니다.

<<!DOCTYPE html>

<head>
<style>
    #id {
        background-color: white;
    }
    
    .class1 {
        font-size:30px
    }

    .class2 {
        color: orange;
    }

    .class3 {
        background-color" red;
    }
</style>
<body>
    <h1 id="id">h1</h1>
    <h2 "class1 class2 class3">h2</h2>
</body>
