CSS 색상

CSS를 통해 색을 표현하는 방법

색상 이름 표기법

색상 이름으로 표현 가능하다.

W3C에서 정의한 16개의 HTML4 표준 색상
black aqua blue green
gray white lime maroon
silver olive red purple
navy teal fuchsia yellow

색상 이름 사용 예시

<style>
    .black { color : black; }
    .white { color : white; }
    .navy { color : navy; }
</style>

RGB 색상값 표기법

R (RED), G(GREEN), B(BLUE) 세 가지 색의 혼합을 표현한다.

부터 255 사이의 범위를 가지며 이를 (R,G,B)로 표현한다.

RGB표현 사용 예시

<style>
    .blue { color: rgb(0,0,255); }
    .green { color: rgb(0,128,0); }
    .silver { color: rgb(192,192,192); }
</style>

16진수 색상값으로 표현

16진수 색상값은 RGB 색상값을 각각 16진수로 변환한 것이다.

RGB 색상의 기본색(Red, Green, Blue)은 각각 00부터 FF까지의 범위를 가진다.

16진수표현 사용 예시

<style>
    .red { color: #ff0000; }
    .white { color: #fff; }
    .silver { color: #C0C0C0; }
</style>
