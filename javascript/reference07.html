반복문

반복문이란 조건이 성립한다면 해당 코드르 원하는 횟수만큼 반복하여 실행할 수 있는 것입니다.

반복문에 종류에는 while문, do while문, for문이 있습니다.

while문

while문은 조건식을 만족할 때까지 여러번 반복할 수 있습니다.

var 변수 = 초기값;
while(조건식){
    자바스크립트 코드;
    증감식;
}
적용예시
<script>
var i = 1;
while(i <= 5) {
    document.write("안녕하세요" + i,"<br>");
    i++;
}
document.write("=== The End ===");
<script>

-결과값-
안녕하세요1
안녕하세요2
안녕하세요3
안녕하세요4
안녕하세요5
=== The End ===

for문

조건식을 만족할 때까지 코드를 반복 하여 실행합니다. 실제 사용 빈도가 높습니다.(기본형)

for(초기값; 조건식; 증감식;){
  코드;
}
적용 예시
<script>
for(var i = 1; i<= 5; i++){
    document.write("반복" + i, "<br>" )
}
<script>

-결과값-
반복1
반복2
반복3
반복4
반복5

break문

반복문인 while문 또는 for문에서 break문을 실행하면 조건식과 상관없이 강제로 반복문을 종료합니다.

즉, 반복문을 강제로 종료할 때 사용합니다.

-for문-
for(초기값; 조건식; 증감식;){
  break; //반복문을 강제로 종료합니다.
  자바스크립트 코드;
}

-while문-
var 변수 = 초깃값;
while(조건식){
  break;
  자바스크립트 코드;
  증감식;
}
적용 예시
<script>
    for(var i = 1; i<= 10; i++){
        if(i == 6)break;
        document.write(i, "<br>" )
    }
    document.write("===The End===");
    <script>
    
    -결과값-
 1
 2
 3
 4
 5
 ===The End=== // i값이 6이되면 break문으로 종료합니다.

continue문

continue문은 반복문에서만 사용할 수 있습니다.

for문에서는 continue 다음에 오는 코드를 무시하고 바로 증감식으로 넘어갑니다.

while문에서는 continue 다음에 오는 코드를 무시하고 바로 조건식으로 넘어갑니다.

-for문-
for(초기값; 조건식; 증감식;){
  continue;
  자바스크립트 코드;
}

-while문-
var 변수 = 초깃값;
while(조건식){
  continue;
  자바스크립트 코드;
  증감식;
}
적용 예시
<script>
    for(var i = 1; i<= 5; i++){
        if(i ==  3) continue;
        document.write(i, "<br>" )
    }
    document.write("===The End===");
    <script>
    
    -결과값-
    1
    2
    4
    5
    ===The End=== // i값이 3이되면 continue문으로 코드 실행을 건너뜁니다.
