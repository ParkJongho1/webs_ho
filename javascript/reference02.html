변수란

  • 프로그램이 실행될 때 잠재적으로 변경 될 수있는 정의 된 값을 말한다.
  • 특정 값을 보관하는 메모리 공간(이름이 붙은 저장소)
  • 각 데이터가 보관되있는 것의 이름이 변수명이다.
  • 변수명(이름)을 통해 데이터를 사용할 수 있다

변수 선언 방법

변수 선언하는 방법

변수를 선언 할 때는 다음과 같이 var, let등 키워드를 변수명 앞에 붙입니다.

var 변수명 = 값;
let 변수명 = 값;

한글 사용이 불가능하고 영문과 숫자, 그리고 _ , $ 만 쓸 수 있음

단어와 단어의 조합이라면 두 번째 글자의 첫 글자는 대문자로 표기 (Camel 표기법)

변수명

자바스크립트에는 의미를 가진 단어들이 있기 때문에 가용자가 임의로 사용할 수 없는 단어들이 있습니다.

키워드 : 자바스크립트에서 사용하는 단어
식별자 : 사용자가 임의로 사용하는 단어

식별자를 만드는 규칙

  • 사용자가 사용할 수 있는 문자 : 알파벳(a~z, A~Z), 숫자(0~9), 밑줄(_), 달러기호&($)
  • 첫글자는 숫자를 사용할 수 없습니다.
  • 공백을 사용할 수 없습니다. : 밑줕(_)사용 가능, 하이픈(-) 사용불가
  • 특수기호를 사용할수 없습니다. : 달러기호($)는 사용 가능
  • 키워드를 사용할 수 없습니다.

    단어를 조합할 때 규칙

  • 언더스코어 노테이션 : (color_top) : 중간에 '_'를 사용합니다.
  • 캐멀 노테이션 : (colorTop) : 두번째 단어에 첫 글자를 대문자로 사용합니다
  • 파스칼 노테이션 : (ColorTop) : 모든 단어의 첫 글자를 대문자로 사용합니다.
