for()문

  • for문은 while문과는 달리 자체적으로 초기식, 표현식, 증감식을 모두 포함하고 있는 반복문입니다.
  • for문을 구성하는 초기식, 조건식, 증감식은 각각 생략될 수 있습니다.
  • 쉼표 연산자(,)를 사용하면 여려 개의 초기식이나 증감식을 동시에 사용할 수 있습니다.
  • for문의 종류에는 for(기본), foEach, for in, for of 가 있습니다.

for()문 문법

for

모든 자료형에 대해 사용이 가능하고 반복문 건너뛰기(continue),종료(break)가 가능하다. 변수 활용이 가능하다.


문법
예시
const arr = [100, 200, 300, 400, 500];

    for( let i=0; i<arr.length; i++ ){
        document.write(arr[i]);
    }
//출력결과 100 200 300 400 500

forEach

코드를 한분에 보기 쉽고 간결하다. 성능 측면에서 매우 준수하다. 여러 기본 매개변수를 통해 활용도가 높다.


문법
예시
const arr = [100, 200, 300, 400, 500];

arr.forEach(function(el){
    document.write(el);
}
//출력결과 100 200 300 400 500

for in

객체에 쉽게 접근하여 반복문을 수핼할 수 있다. 반복문 건너뛰기(continue),종료(break)가 가능하다. 


문법
예시
const arr = {
    a: 1,
    b: 2,
    c: 3
};

for ( let i in arr) {
    document.write(i, arr[i]);
}
//출력결과 a 1, b 2,c 3

for of

반복 가능한 객체에 대해 간결하게 반목문을 작성 가능하다. 반복문 건너뛰기(continue),종료(break)가 가능하다.


문법
예시
const str = 'hello';

for ( let s of str) {
   document.write(s);
}
//출력결과 h, e, l, l, o
